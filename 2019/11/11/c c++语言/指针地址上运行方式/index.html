c c++语言/指针地址上运行方式

​

​ 在c语言中，定义了一种在地址上运行的方式，我们已经有了上面的语句， 这时候如果通过printf(“%d”, *(p + 1)); 或者printf(“%d”, *(&a[0] + 1)); 进行输出，我们都会得到a[1]元素的值，因为p + 1和&a[0] + 1这中计算实际上都会得到a[1]的地址。

​ 类似的，p + 2、&a[0] + 2 或 &a[1] + 1都会得到 a[2] 元素的地址。
