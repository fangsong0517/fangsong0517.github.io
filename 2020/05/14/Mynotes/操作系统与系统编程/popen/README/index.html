主要通过pipe管道实现，具体思路如下：

​ 1、使用pipe()建立管道

​ 2、使用fork()创建子进程

​ 3、在子进程中调用exec族函数执行命令，通过管道将结果传送至父进程

​ 4、在主进程中等待子进程执行，子进程执行完成后将接收其结果，返回结果的文件指针
