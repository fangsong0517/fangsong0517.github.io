平衡二叉查找树

二叉排序树

名称：二叉排序树、二叉搜索树
性质: 
     1、左子树 < 根节点
     2、右子树 > 根节点
用途:
     解决与排名有关的检所需求

二叉查找树插入

例如插入10, 10 < 20插入到左子树，10<17插入到左子树,10 > 3插入到右子树，右子树为空放置10

二叉查找树的删除

1、删除叶子节点
2、删除出度为1的结点
3、删除出度为2的结点

(1)删除出度为1的结点，提升３的唯一子树
